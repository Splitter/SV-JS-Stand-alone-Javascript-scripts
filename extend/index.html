SV Extend Menu extending menu script

> Scrolling Backround Menu Script

This script will allow you to create a fancy menu effect where as an item extends in length as a user hovers over a menu item. It is completely stand alone requiring absolutely no third party frameworks or libraries. And is less then 1kb minified.

Examples:


Horizontal Extend Menu

  • Home
  • About
  • Portfolio
  • Contact
  • Home
  • About
  • Portfolio
  • Contact


The Code:


extend.init('nav_container','h',NewX,NewY,Speed,Steps);
extend.init('nav_container2','v',NewX,NewY,Speed,Steps);


Must be called on window.onload and not when the dom loaded, it must be initialized after the images have loaded.
